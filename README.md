# Ella-frontend (Chrome Extension)

Ella-frontend is a Chrome extension designed to work seamlessly with the **Hakuneo AI** website. It acts as a **prompt assistant**, helping users create better prompts and interact more effectively with Hakuneo AI.

## How to Run & Build

1. Clone the repository  
   ```bash
   git clone https://github.com/<your-username>/Ella-frontend.git
   cd Ella-frontend
   ```

2. Install dependencies  
   ```bash
   npm install
   ```

3. Build the extension  
   ```bash
   npm run build
   ```

   This will create a `./dist` folder containing the final build.

4. Load the extension in Chrome  
   - Open Chrome and go to **chrome://extensions/**
   - Enable **Developer mode** (top-right corner)
   - Click **Load unpacked**
   - Select the `./dist` folder generated by the build command  

5. The extension is now ready to use. Open the **Hakuneo AI** website and the Ella extension will assist you by improving prompts and providing smart suggestions.
